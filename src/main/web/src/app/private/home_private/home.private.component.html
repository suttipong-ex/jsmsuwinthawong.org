บันทึกคำขอ(มีค่าธรรมเนียม)
      แบบฟอร์ม ก.01 คำขอจดทะเบียนเครื่องหมายการค้า เครื่องหมายบริการเครื่องหมายรับรองหรือเครื่องหมายร่วม
      แบบฟอร์ม ก.02 คำคัดค้านการขอจดทะเบียนเครื่องหมายการค้า
      แบบฟอร์ม ก.03 คำอุทธรณ์
      แบบฟอร์ม ก.04 คำขอโอนหรือรับมรดกสิทธิ
      แบบฟอร์ม ก.05 คำขอจดทะเบียนสัญญาอนุญาตให้ใช้เครื่องหมายการค้า/บริการ
      แบบฟอร์ม ก.06 คำขอแก้ไขเปลี่ยนแปลงรายการจดทะเบียน
      แบบฟอร์ม ก.07 คำขอต่ออายุการจดทะเบียน
      แบบฟอร์ม ก.08 คำขอให้เพิกถอนการจดทะเบียน
บันทึกคำขอ(ไม่มีค่าธรรมเนียม)
      แบบฟอร์ม ก.02 คำโต้แย้ง
      แบบฟอร์ม ก.10 คำขอถือสิทธิวันที่ยื่นคำขอนอกราชอาณาจักรครั้งแรกหรือวันที่นำสินค้าที่ใช้เครื่องหมายออกแสดงในงานแสดง สินค้าระหว่างประเทศเป็นวันยื่นคำขอในราชอาณาจักรตามมาตรา 28 หรือมาตรา 28 ทวิ
      แบบฟอร์ม ก.12 หนังสือแสดงการปฏิเสธ
      แบบฟอร์ม ก.14 หนังสือแจ้งฟ้องคดีต่อศาล คำพิพากษาและผลคดี
      แบบฟอร์ม ก.15 หนังสือชี้แจงการถูกร้องขอให้เพิกถอนการจดทะเบียน
      แบบฟอร์ม ก.19 หนังสือขอผ่อนผันการส่งหลักฐาน
      แบบฟอร์ม ก.20 หนังสือนำส่งเอกสารหลักฐานและคำชี้แจง
บันทึกรายการชำระเงิน(อื่นๆ)
      บันทึกการชำระเงิน บันทึกการชำระเงินที่ได้รับการอนุมัติ
